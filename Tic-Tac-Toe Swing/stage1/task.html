Description

Tic-tac-toe is known all over the world. Each country may have its own version of the name, sometimes the rules are different, but the essence of the game remains the same. Below are the main rules that may be familiar to you since childhood.

Tic-tac-toe is a game played by two players on a 3x3 grid. One of the players is 'X', and the other player is 'O'. X plays first, then O takes the next turn, and so on. The players write 'X' and 'O' on a 3x3 field. The first player that puts 3 X's or 3 O's in a straight line (including diagonals) wins the game.

In this project, you will create a Tic-Tac-Toe game with a graphical user interface (GUI) using the Swing framework. The program will allow you to play both against another player or against the computer. You can also configure the game so that the computer plays against itself, and you will observe the process of the game.

Objectives

Your first task is to create the main window of the program and the field for the game. In Swing, the class that represents an empty window is JFrame. To improve the basic window, you should extend this class and write your own logic for the window. Here is an example of code in the constructor of the class that extends JFrame:

setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
setTitle("Tic Tac Toe");
setResizable(false);
setSize(450, 450);
setVisible(true);

In addition to creating a window, you need to create a field with nine cells, which we will place on a field in three rows and three columns. For cells, we can use a class that extends the JButton class. For testing purposes, you must give a name to each of the cells using the setName method. We will use algebraic notation just like in chess. The bottom left cell will be named “ButtonA1” and the top right “ButtonC3”. Also, at the first stage for the text, we will indicate the coordinates of these cells from "A1" to "C3". After starting the program, it should look like an illustration.

To create the field itself, we can use a separate Board class extending JPannel. You can use GridLayout (3, 3) as a Layout. Then you need to create nine cells and add them to this class. The Board class itself needs to be added to the main window with the TicTacToe class.
