Description

At this stage, we will complete our program. We will add a menu to quickly launch the game in a specific player configuration. We will also add a more detailed message in the status bar about the current turn and at the end of the game.

Objectives

We will make the messages in the status bar more informative. Now, together with the status “The game is playing”, the program will show which player is the move. The message should look like: “The turn of {0} Player ({1})”. After the end of the game, the program should print in the status line: “The {0} Player ({1}) wins”. We have provided messages in the format of the MessageFormat class. Here, instead of {0} should be the player's name “Human” or “Robot”, and instead of {1} should be “X” or “O”.

You also need to add a Game menu containing five items:

  1. Human vs Human
  2. Human vs Robot
  3. Robot v Human
  4. Robot vs Robot
  5. Exit

You can add a dividing line before the “Exit” menu item. This is done using the code:

menuGame.addSeparator();
menuGame.add(menuItemExit);

When you select one of the first four menu items, the program must clear the field, set the desired values ​​for the players in the toolbar and start the game. Please note that when starting the game, the "Start/Reset" button must be set to "Reset" mode, the players' buttons must be disabled, and the field cells must be enabled.

For testing purposes, the menu components should be named as follows:

  • menuGame
  • menuHumanHuman
  • menuHumanRobot
  • menuRobotHuman
  • menuRobotRobot
  • menuExit
